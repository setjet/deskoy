Quick fixes & shortcuts

  • Cover is showing but the hotkey won’t respond (focus quirks, fullscreen games, etc.)? Press Esc—it always dismisses the cover.
  • This settings window is open? Esc closes it too when Deskoy has focus.
  • Edit jumps to the main window so you can re-record the hotkey; save from the toolbar there when you’re done.
  • Need the shortcut free for another app? Pause Deskoy above unregisters the hotkey until you resume (then Deskoy registers it again).
